# 🐄 Lumpy Skin Disease Detection using Deep Learning (ResNet50)

## 📌 Overview
This project is an AI-based system that detects Lumpy Skin Disease (LSD) in cattle using image classification.  
It helps farmers identify infected animals early and take necessary action.

---

## 🚀 Features
- Detects **Healthy vs Infected** cattle images
- Built using **ResNet50 Deep Learning model**
- User-friendly **GUI interface**
- Supports **Image Upload & Live Camera**
- Shows **Prediction + Confidence Score**
- Provides **AI Assistant (Symptoms, Care, Prevention)**
- Supports **English & Telugu languages**

---

## 🧠 Technologies Used
- MATLAB
- Deep Learning (ResNet50)
- Image Processing

---

## 📊 Model Performance
- Accuracy: 94
- Evaluated using **Confusion Matrix**
